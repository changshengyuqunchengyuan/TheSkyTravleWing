
地理设定
==============
1、【游牧部落】
--------------
北方游牧民族：由近百个大小部落组成，居住在大陆北部广袤的草原上。以放牧为生，随水草而栖。因此部落之间会为水、草而 互相征伐吞并。信仰原始教义：即自然崇拜、图腾崇拜、祖先崇拜和萨满信仰。

2、【冰原火山：冰原狼的地盘】
--------
冰原狼：生存在冰原的体型硕大的狼群，与游牧民族毗邻
而居。狼群由一只智慧超人的独眼白狼领导，游牧民族称之为白狼王， 没人知道它活了多久，也鲜少有人见过它的真实面目，亦不知雌雄。传说神魔时代白狼王曾经与人订契约共同战斗。不败的狼 骑兵团就是由此而来。

3、【冰原火山和游牧部族中间】
--------
A国：大陆中心大帝国。拥有强大的军事、肥沃的土地、灿烂的文化和繁荣的经济。可以说是世界的中心。但是仍然饱受北部游 牧民族和漠北马贼的侵扰。因民族众多，地域宽广处大陆中心是各个国家间来往的通道，因此拥有种类繁多的信仰。

4、【漠北马贼】
--------
这群游走于帝国西北戈壁滩和西海岸的家伙多是大陆各处的流民、罪犯和“冒险者”。在他们眼中只有钱。简而言之 ，就是一群亡命狂徒。
5、【青之裂谷】
--------
神魔时代一度传说的地方，由于常年没有人涉足，整座山上长满青色青苔，环境之险恶以至于无人敢攀爬，再加上山谷中间莫名出现的一道深厚裂缝，看起来像是被一刀劈开，极其骇人。
6、【龙腾山谷】
--------
传说中巨龙栖息的地方，传言中，原本是没有这座山谷的，但是在神魔时代的某一天，有人看见一条金色巨龙盘旋而下，坐落于此地，有贪婪的人欲杀死巨龙，取其金麟及龙血等，却被金龙封印到地底，这些人的贪婪之血污染了这片土地，金龙就这么盘起身子，坐落于此地，由此形成一座山谷，由于常年有人看见山上有类似于龙的生物飞过，于是命名为龙腾山谷。
7、【虫巢】
--------
在这片大陆上，虫子原本是一种最为低贱的生物，人随意便可杀死，于是有一只甲虫，为了洗刷所有虫类的耻辱，与一个修炼千年的老者签订契约，将整个虫族的灵魂做抵押，得到了老者的力量，从而将所有虫类化为巨大生物，他们慢慢建起自己的领地，并在整个大陆上获得一席之地。
8、【古语森林】
--------
传说在神魔时代之前，古语森林就屹立于属于他的土地上了，神魔时代时有神族的战士路过这里，认为自己是神便进入了里面，在里面时遇到大批超强魔兽，不敌，被捕，由于魔兽也有高智商，他们让这个战士为他们永远效劳，神族的战士在这里苟延残喘，大天使得知后愤怒不已，想用雷电毁灭这片森林，但是雷电却被古语森林外的一道黑色防护罩轻松挡下，有人在那时听见森林中传来一种失传千万年的古语，就此，古语森林的名字立刻传播在了整个大陆上。
9、【无主之地】
--------
无主之地，是一片资源丰富，却不曾有军队来占领的地方，原因很简单，这片土地上，有一大批实力绝对可以以一敌百的魔兽，而且数量之多令人难以想象，令人望而却步。传闻中这里曾经的主人是堕落天使斯林克，但是堕落天使被大天使主神杀死后，所有魔兽疯狂起来，他们自相残杀，但数量却不曾减少，戾气越来越浓厚，最后导致这块宝地永远无主。
10、【逝者沙漠】
--------
这片沙漠曾经是神魔大战时，两军交战的主要地点，军队在这里厮杀着，死者不计其数，但是，惨无人道的两边将军，抛弃了上亿士兵尸体，不做处理，亡者的鲜血覆盖了这片土地，所有阻挡风沙的树木瞬间腐化，沙尘暴袭击着这篇地方，最终形成沙漠，来到这里，随意往地下一挖，就能看见一句骷髅，来过这里的人，都称他为逝者沙漠。
11、【永恒之碑】
--------
在逝者沙漠以及远古遗迹中间，有一处地点，那是一块黑色石碑，上面刻着古语，中间则放着一把已经石化的长剑，传说中，由于逝者沙漠中的亡灵力量太强，以及远古遗迹中的怨恨之力太大，大天使只能用逝者沙漠中的一部分鲜血，凝固，做成了永恒之碑，封印住两股力量不相容，并放上自己的天使之剑作为献祭，但是终有一天，这个封印，会被打破……
12、【远古遗迹】
--------
远古时代，神和魔刚刚诞生，他们都在远古遗迹中出现，相遇不可避免，最开始，他们如同人类一样，是很好的朋友，但慢慢的，后来出现的魔开始歧视神，被神知道后，他们也开始敌视魔，最终，悲剧发生了，无法计量的神和魔以不同的处事方法开始互相厮杀，最终只剩下后来的大天使以及魔鬼拉克丝，他们便是最初的神魔，最终，因为怨恨而死的二人产生了极大的怨念体，而后来出现的大天使，则铸造了永恒之碑，来封印他们……
13、【断界黑塔】
--------
神魔时代最后的一段时间，大天使再也忍受不了这般杀戮，于是用各个地方的亡者鲜血，铸成一座巨塔，并且将大陆一分为二，从这座塔中最中间的地方向两边看去，一边是白天，一边是黑夜，中间的分割线相当清楚，由此名为断界黑塔。
14、【极北之地】
--------
与大陆隔开的一块小地方，但却寒冷无比，传闻是恶魔在这里的地下放置了一块巨大的恶魔寒冰，千年不化，于是这里由于过于冰冷，影响到的地方和大陆分离开来，最后的结果就是变成一个极其偏僻的地方。
15、【鱼人族领地】
--------
与虫族一样，鱼类之前一直被人类当成食物，但是，生性并不好战的鱼族向大天使祈祷，让他们变成人类无法羞辱的生物，于是大天使将他们变为了一种和人差不多的生物，但是他们只有人类的体态大小和四肢，身体依然是鱼身，他们并不选择侵略人类，只是用这种力量保护自己，并建立起了自己的领地。
16、【双子山】
--------
这座山相当诡异，山上最顶峰的地方有一层看不见的魔法屏障，凡事碰过这个屏障的人，都会看见在自己的对面，多出一个“自己”，并且对方肯定会叫自己“哥哥”或是“姐姐”，但是对方却又无法跨越屏障，却会使来过的人来过的人迷上这里，永远守候在这里，所以被称作双子山。

17、【铁之森林与红石谷中间】
--------
西大陆的B国：紧靠着繁茂的原始森林和矿石丰富的红石谷，自然资源丰富，与A帝国是同盟，贸易往来频繁。B国的贸易商队马 贼眼中的大肥羊，有时还和鱼人族的海盗
在海上抢劫商队。

18、【铁之森林与无主之地中间】
--------
西大陆C国：土地贫瘠资源匮乏的国度，南部大片的荒芜之地，人迹罕至。穿越过无主之地--逝者沙漠这条死亡走廊，就能到达 传说中的神魔时代的远古遗迹，遗迹中埋葬着大量的财宝，令探险者们趋之若鹜。据说神魔时代，此处并不似今天这样荒凉，由于 魔族强弩之末之时，携带着大量的财宝，准备通过此地坐船逃离到迷雾洋中的群岛上，但是很快就受到了人类的联军阻击，发动了史上最为惨烈的战斗，这场战斗彻底摧毁了这块本来丰沃繁荣的土地。

19、【冥空神殿】
--------
翼族是整个大陆上最为古老的种族之一，孤傲的个性牵动了他们时代发展，他们并不与人族或者其他种族经常交涉，但是他们依然拥有使别的种族不敢轻易侵略的力量，原因就是——他们飞行的力量，在空中他们可以发动高于地面攻击时二十倍力量的攻击，所以，这是个可怕的种族。而他们的领地，则是利用万千翼族的羽毛铸成的神殿！
20、【黑旗海贼】
--------
这是一支原本毫无名气可言的海贼组织，但是，在某次抢劫商船时，他们无意见得到了A国国王的宝物：雷神之钻，于是他们威胁了D国国王，得到了D国的一块国土后，他们肆意招收海贼，慢慢壮大，最后变得令人闻风丧胆。
21、【云海与幽之海】
--------
云海原本是幽之海的一部分，但是，在神魔时代的时候，魔族为了抢先一步争夺国土，企图杀死神族地界管理员，大天使为了保护土地和下属，迫不得已用神力，把现今的云海上移，移动到了幽之海上方，导致幽之海常年无法得到阳光照射，永远幽暗。
